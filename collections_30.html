キュー

Scala は不変キューの他に可変キューも提供する．mQueue は不変のものと同じように使うが，enqueue の代わりに += と ++= 演算子を使って加算する．また，可変キューでは dequeue は先頭の要素を削除して，それを返す．次に具体例で説明する．

scala> val queue = new scala.collection.mutable.Queue[String]
queue: scala.collection.mutable.Queue[String] = Queue()
scala> queue += "a"
res10: queue.type = Queue(a)
scala> queue ++= List("b", "c")
res11: queue.type = Queue(a, b, c)
scala> queue
res12: scala.collection.mutable.Queue[String]
  = Queue(a, b, c)
scala> queue.dequeue
res13: String = a
scala> queue
res14: scala.collection.mutable.Queue[String] = Queue(b, c)
